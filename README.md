vim-win-seek
============

Seaching in files for Windows. Approximately 10 times faster than vimgrep.

Installation
------------

Use Vundle (otherwise may be problems with path to exe):<br/>
	<code>Plugin 'https://github.com/cser/vim-win-seek.git'</code>

Usage
-----

Type in vim:<br/>
	<code>:help seek</code>
